#### 1 时间复杂度

一个算法语句总的执行次数是关于问题规模N的问题
N 为问题的规模
T（N） 为语句总的执行次数
算法的执行次数的增长速率和f(N)的增长速率相同 T(N)=O(f(n))
O(f(n)) 时间复杂度的O渐进表达法
